#  Exploratory Data Analysis (EDA) on Bank Marketing Dataset

##  Project Overview

This project performs an **Exploratory Data Analysis (EDA)** on the Bank Marketing Dataset to understand the data, identify patterns, detect anomalies, and extract meaningful insights before applying machine learning models.

The analysis includes data exploration, cleaning, visualization, and statistical analysis using Python.

---

##  Objectives

- Understand the structure of the dataset.
- Explore numerical and categorical features.
- Handle missing values (if any).
- Detect duplicate records.
- Identify outliers using the IQR method.
- Analyze feature distributions.
- Study relationships between variables.
- Generate meaningful business insights through visualizations.

---

##  Technologies Used

- Python
- VS Code
- Pandas
- NumPy
- Matplotlib

---

##  Dataset

**Dataset:** Bank Marketing Dataset

The dataset contains customer information collected during direct marketing campaigns conducted by a banking institution. The goal is to analyze customer characteristics and campaign-related information to better understand the data.
